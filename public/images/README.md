## Yêu cầu hình ảnh
- Hình ảnh sản phẩm: Kích thước tối thiểu 800x800px, định dạng JPG/PNG
- Hình ảnh đại diện: Kích thước tối thiểu 200x200px, định dạng JPG/PNG
- Tối ưu hóa hình ảnh trước khi upload để giảm dung lượng
